#My Exel
This project is a simple Exel simulator in C.
it allows user to manage matrices, perform operations on cells,and maintain data in a structured way.
##Features
-Creats, manage, and display matrices
- Add or delete rows and columns
- Perform mathematical operations on matrix elements
- Prioritize or modify data
- Persistent storage through `database.txt`
Project Structure
expression.c / expression.h : Handle expressions and calculations
file_operation.c / file_operation.h : Reading/writing data to files
linked_list.c / linked_list.h : Linked list implementation for matrices
settings.h : Program configuration and constants
user_interface.c : Handles user input and output
database.txt : Persistent storage for data
